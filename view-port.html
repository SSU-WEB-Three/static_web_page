HTML & CSS 학습 자료
웹 개발 기초 자료
  • MDN HTML 문서 - HTML의 기초부터 고급 기능까지 자세히 다루고 있는 자료입니다.
  • W3Schools - HTML, CSS, JavaScript 등 다양한 웹 기술을 쉽게 배울 수 있는 사이트입니다.
  • CSS-Tricks - CSS와 관련된 팁과 트릭, 다양한 예제를 제공하는 블로그입니다.
  • freeCodeCamp - HTML, CSS, JavaScript를 포함한 다양한 웹 개발 과정이 제공되는 무료 온라인 학습 사이트입니다.
추천 강의 및 온라인 코스
  • The Web Developer Bootcamp (Udemy) - 웹 개발을 위한 전반적인 강의로, HTML과 CSS를 포함한 기본 기술들을 배울 수 있습니다.
  • Web Design for Everybody (Coursera) - 웹 디자인과 개발의 기초부터 배우고, HTML, CSS, JavaScript를 포함한 기본적인 기술을 익힐 수 있습니다.
기타 유용한 리소스
  • FontAwesome - 다양한 아이콘을 제공하는 라이브러리입니다.
  • Pexels - 무료로 고해상도 이미지를 제공하는 사이트입니다.
  • Unsplash - 무료 고해상도 이미지를 제공하는 사이트입니다.
